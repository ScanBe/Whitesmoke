<!--...-->

Defines a comment

<!--This text will be ignored by the browser-->

Comments are most useful for readibility and documention of your process/steps.

<!DOCTYPE>

Defines the document type

This is most commonly used to denote which version of HTML is being utilized.

<!DOCTYPE html> denotes the use of HTML5.

There is no closing tag

<a>

Defines a hyperlink

<a href="www.google.com">Click Me</a> → Click Me

This is used to link another element, page, or event.

It is most commonly used for external links and navigation accross your website.

<abbr>

Defines an abbreviation or an acronym

<p>We are learning about<abbr title="Hyper Text Markup Language" >HTML.</abbr></p> → We are learning about HTML.

Denotes the use of an abbreviation and allows the full title to show when hovered, if stated.
